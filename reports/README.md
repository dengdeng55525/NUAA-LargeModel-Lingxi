# Reports

This directory stores generated experiment records, comparison reports, and screenshots for the LoRA course project.

Expected generated files:

- `before_after_compare.md`: base model vs. LoRA adapter response comparison.
- `domain_qa_eval.md`: domain QA comparison across the base model and LoRA adapters.
- `dpo_domain_qa_eval.md`: base vs. SFT vs. DPO domain QA comparison.
- `dpo_alignment_report.md`: second-stage DPO preference alignment summary.
- `hyperparameter_analysis.md`: rank, training-step, and metric summary for the ablation experiments.
- `memory_module_design.md`: short-term emotion memory module design and usage.
- `neftune_experiment_design.md`: NEFTune training enhancement design and report-writing notes.
- `rslora_experiment_design.md`: rank-stabilized LoRA experiment design and report-writing notes.
- `safety_boundary_design.md`: safety boundary and refusal-rule design for high-risk emotional inputs.
- `training_summary.md`: loss curves, rank/epoch ablation notes, and representative examples.
